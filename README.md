# JavaScript/React ES6 Practice Project – 25 Days Challenge 🚀

A structured 25-day coding challenge to build a **strong JavaScript foundation** and advance into **React + ES6** mastery through hands-on daily projects.

---

## 📅 PHASE 1: Strong JavaScript Foundation (Day 1–7)

### ✅ Day 1 – Variables, Data Types, Operators

**Concepts Covered:**
- `let` vs `const` — block-scoped variable declarations
- Primitive types (`number`, `string`, `boolean`, `undefined`, `null`) vs Reference types (`object`, `array`)
- Arithmetic operators (`+`, `-`, `*`, `/`, `**`, `%`)
- Comparison operators (`===`, `!==`, `>`, `<`, `>=`, `<=`)
- Input type validation using `typeof`, `isNaN()`, `isFinite()`

**Projects Built:**
| # | Project | File | Description |
|---|---------|------|-------------|
| 1 | 🌡️ Temperature Converter | `temperatureConverter.js` | Converts Celsius ↔ Fahrenheit with input validation |
| 2 | ⚖️ BMI Calculator | `bmiCalculator.js` | Calculates BMI (metric & imperial) with WHO classification |
| 3 | 🖥️ Interactive UI | `index.html` | Visual interface for both calculators (open in browser) |

**Stretch Goals:**
- ✅ Input type validation on all functions
- ✅ Handles edge cases: `NaN`, `Infinity`, `null`, `undefined`, negative numbers
- ✅ Sanity checks for unrealistic values

## 🛠️ How to Run

```bash
# Navigate to day folder
```
